# **App Name**: Cosmic Viewer

## Core Features:

- 3D Solar System Rendering: Render a 3D solar system using Three.js, including the Sun and eight planets.
- Planet Textures: Apply realistic textures to each planet.
- Orbital Mechanics: Simulate planet orbits and rotations with adjustable speed and pause/play controls.
- Interactive Controls: Allow users to rotate, zoom, and pan the solar system using OrbitControls.
- Planet Information Cards: Display planet information (name, diameter, distance, orbital period, description) in UI cards upon selection.
- UI Control Panel: Provide a control panel with toggles for labels, orbit paths, rotation speed, and camera auto-rotation.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to evoke a sense of deep space.
- Background color: Very dark desaturated blue (#1A222D) for contrast and depth.
- Accent color: Bright yellow (#F39C12) to highlight interactive elements and call attention.
- Font: 'Inter', a sans-serif font suitable for both headlines and body text due to its modern and neutral look.
- Use simple, white line icons for UI controls to maintain a clean and modern aesthetic.
- Employ a responsive layout with Tailwind CSS, optimizing for both desktop and mobile devices. Ensure key information remains accessible across screen sizes.
- Incorporate smooth transitions and animations using Framer Motion for UI elements, such as opening planet information cards and toggling settings.
